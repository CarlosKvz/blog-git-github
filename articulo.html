Git Branch

Con git branch + "nombre de la rama" puedo crear una rama en git que copia los archivos actuales de main, para posteriormente trabajar con ellos de manera independiente, eventualmente puedo unir estos archivos con git merge.

Git Merge

Este comando nos permite unir dos ramas en nuestro repositorio, funciona como un commit. Si existen modificaciones a las mismas lineas de codigo en ambas ramas, revisamos estos cambios y elijimos cuales deben quedar para posteriormente hacer el commit para realizar el merge.
